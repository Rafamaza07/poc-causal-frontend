Cómo usarlo en tu web (pásaselo a Sonnet)

<!-- 1. En el <head> de tu web -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@600&display=swap" rel="stylesheet">

<!-- 2. Para usar el SVG (lo más simple): -->
<img src="/logo/kausalia-logo.svg" alt="KausalIA" height="40">

<!-- 3. Como favicon: -->
<link rel="icon" href="/logo/kausalia-favicon.svg" type="image/svg+xml">

<!-- 4. Para calidad máxima: pega el componente HTML
     (ver archivo kausalia-logo.html) y ajusta tamaño con
     style="--logo-size: 80px" -->
Notas:
• Tipografía: Inter 600 (gratis, Google Fonts) · Color gris #2B2F36 · Degradado azul #0E1A44 → #1E3A8A → #3B7BD9.
• Área de seguridad mínima: igual a la altura de la letra “K”.
• Si necesitas un SVG con la tipografía convertida a curvas (sin depender de Google Fonts), dímelo y lo preparo.
